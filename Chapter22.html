Sets

Python set

  • A Python set is an object that stores a collection of data in the same way as a mathematical set
  • All items must be unique(no duplicates)
  • Set is unordered (insertion order not maintained)
  • Elements in a set can be of different data types
  • A set can be modified, but the elements of a set are immutable (since there is no indexing in set)

Creating a set

  • Method 2: a set can be defined with curly braces({}):
    • For example, x = {'x',q','u'}
    • if we call print x, we get:{'x,'q','u'}(notice how duplicates are removed)
    • However, x ={} creates an empty dictionary. Hence the only way to create an empty set is to call set ()

set built in functions

Function Description
len(set1) Returns the number of elements in the set1.
add(element) Adds an element to the set
update(iterable) Adds multiple elements from an iterable to the set
pop() Removes and return a random element
clear() Clears all elements in the set
remove(element) Removes the specified element(Raises a KeyError exception if element is not in the set)
discard() Removes the specified element(Does not raise an exception)

set operations

Operation Description
set1.union(set2)
OR
set1 | set2 		Combines all unique element from set1 and set2
set1.intersection(set2)
OR
set1 & set2 		Returns elements common between set1 and set2
set1.difference(set2)
OR
set1 - set2 		Return elements in set1 but not in set2
set1.symmetric_difference(set2)
OR
set1 ^ set2 		Returns elements in either set1 or set2 but not in both
set1.issubset(set2)
OR
set1 <= set2 		Checks if all element of set1 are in set2
set1.issuperset(set2)
OR
set1 >= set2 		check if all elements of set2 are in set1
