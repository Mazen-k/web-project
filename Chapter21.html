Dictionaries

dict data type

  • Objects of type dict contain values like lists excpet that we index them uisng keys
  • Think of a dictionary as a set of key-value pairs
  • To create an empty dictionary use:
    • D = {} or D = dict()
  • Sytanx to define a dictionary with specific key-value mappings
    • D = {key1: value1 , key2: value2, ...}
  • Elements in a dictionary are unsorted and cannot be acces with an index
  • Dictionaries are mutablebut keys must be immutable

Operations/Queries on dictionaries

  • Search for a key:
    • key in d: returns True or False
  • Acces value of a key
    • D[key] : gives an error if key is not in D
  • Delete a key
    • del D[key]: gives an error if key is not in D
  • Add new value for a key
    • D[key] = value

More Built in functions

Function Description
len(D) Returns the number of keys in the D.
D.get(key, default) Gets the value associated with given key. If the key is not in D it returns the default msg
D.pop(key, default) Deletes the key from D and return its value.If the key is not in D it returns the default msg
D.keys() Returns all keys in D as a sequence
D.values() Returns all values in D as a sequence
D.copy() Copies the content of D
