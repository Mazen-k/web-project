Strings

str data type

In Python, a string is a sequnce of characters enclosed in single or double quotes

  • str1 = "Sam"
  • str2 = 'Sam'

str indexing

  • Characters can be accesed by specifying the index
    • Non-negative indices: starting from 0 and end at length -1
    • Negative indies: from right to left (reverse order)
  • Strings are immutable data structures meaning they can NOT be modifed
