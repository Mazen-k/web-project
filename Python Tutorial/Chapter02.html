Objects and Data Types

Elements in a python program are called Objects
Objects have specific data types

Data types

int type:

  • Used to represent integers
  • e.g 12,-12,143

float type:

  • Stands for floating point represenation
  • used to represent real num,bers up to a certain precision

bool type

  • Used to represent Boolean/ logical values
  • True and False
  • originates from logical tests

str type

  • Sequence of characters
  • Use single or double quotationsc: "Sam" or 'Sam'
  • more about string in chapter 1.3

Casting

Casting refers to coverting a variable from one data type to another
