Sets

Python set

  • A Python set is an object that stores a collection of data in the same way as a mathematical set
    • All items must be unique(no duplicates)
    • Set is unordered (insertion order not maintained)
    • Elements in a set can be of different data types
    • A set can be modified, but the elements of a set are immutable (since there is no indexing in set)

    Creating a set

    • Method 2: a set can be defined with curly braces({}):
      • For example, x = {'x',q','u'}
      • if we call print x, we get:{'x,'q','u'}(notice how duplicates are removed)
      • However, x ={} creates an empty dictionary. Hence the only way to create an empty set is to call set ()

    Elements in a set

    • len(set) function: returns the number of elements in the set
    • add(element) method: adds an element (of any type) to a set
    • update(argument) method: adds a group of elements to a set
      • argument must be a sequence containing iterable elements, and each of the elements is added to the set
    • pop() method: removes and returns an arbitrary (random) set element.
    • clear() method: clears all the elements of the set
    • remove(element) and discard(element) methods are used to remove the specified element from the set
      • The item that should be removed is passed as an argument
      • Behave differently when the specified item is not found in the set
        • remove method raises a KeyError exception
        • discard method does not raise an exception

    Using the for loop, in, and not in operators with a set

    • A for loop can be used to iterate over elements in a set
      • General format: for item in set
      • The loop iterates once for each element in the set
    • The in operator can be used to test whether a value exists in a set
      • Similarly, not in can be used to test whether value does not exist in the set

    Finding the union of sets

    • Union of two sets: a set that contains all the elements of both sets
    • To find the union of two sets, use the union method
      • Format: set1.union(set2)
    • Or use the | operator
      • Format: set1 | set2
    • Both techniques return a new set that contains the union of both sets

    Finding the intersection of sets

    • Intersection of two sets: a set that contains all elements that exist in both sets
    • To find the intersection of two sets, use the intersection method
      • Format: set1.intersection(set2)
    • Or use the & operator
      • Format: set1 & set2
    • Both techniques return a new set that contains the intersection of the sets

    Finding the difference of sets

    • Difference of two sets: a set that contains the elements that exist in the first set but not in the second set
    • To find the difference of two sets: Use the difference method
      • Format: set1.difference(set2)
    • Or, use the - operator
      • Format: set1 - set2
    • Both techniques return a new set that contains the differnece of the sets

    Finding the symmetric difference of sets

    • Symmetric Difference of two sets: a set that contains the elements that are not shared by the two sets. In other words, the union of (set1 - set2) and (set2 - set1)
    • To find the symmetric difference of two sets, use the symmetric_difference method
      • Format: set1.symmetric_difference
    • Or use the ^ operator
      • Format: set1 ^ set2
    • Both techniques return a new set that contains the symmetric difference of both sets

    Finding Subsets and Supersets

    • Set A is subset of set B if all the elements in set A are included in set B
    • To determine whether set A is subset of set B, use the issubset method
      • Format: setA.issubset(setB)
    • Or, use the <= operator
      • Format: set1 <=; set2
    • Both techniques return True or False based on whether setA is subset of setB or not
    • Set A is superset of set B if it contains all the elements of Bin set A are included in set B
    • To determine whether set A is subset of set B, use the issuperset method
      • Format: setA.issuperset(setB)
    • Or, use the >= operator
      • Format: set1 >= set2
    • Both techniques return True or False based on whether setA is a superset of setB or not
