Variables & Operators

Arithmetic Operations

operators used for int and float:

  • Addition(+): x + y
  • Subtraction(-): x - y
  • Multiplication(*): x * y
  • Division(/): x / y
  • Power(**): x ** y

operators specific for int :

  • Integer Divison(//): x // y is the quotient of x / y
  • Modulo(%): x % y is the remainder of x / y


Relational Operations

operations used to compare two values and return a bool

  • Equal(==): x == y
  • Not equal(!=): x != y
  • Less than(<): x < y
  • Greater than(>): x > y
  • Less than or equal(<=): x <= y
  • Greater than or equal(>=): x >= y

Logical Operation

operations that use boolean logic

  • AND(and): x and y
  • OR(or): x or y
  • NOT(not): not x

Truth tables:

x y x and y x or y
False False False False
False True False True
True False False True
True True True True

Expressions

An expression is made up of operators, values and/or variables

Usally expressions are orderd with parenthesis

If we skip parenthsis order of precednece will be followed from low to high:

1 Exponentiation(**) 5 Relational operators
2 Unary operator minus(-) 6 not
3 Multiplication and division 7 and
4 Addition and Subtraction 8 or

Variables and Assignment operator

In python, a vriable is just a name associated with an object

We use the = operator to intialize a variable

Important remark: the = operator is not an equation
ex: x = x + 1
Python has dynamic data-typing (not a strongly-typed language like C/C++ or Java)
Variables can change data-type
Operations are evaluated at run time for compatibility with objects
Python has rules for variable names
Cannot start with a digit
Cannot contain spaces or symbols
Cannot be a reserved word

List of reserved words:

False class finally is return
None continue for lambda try
True def from nonlocal while
and del global not with
as elif if or yield
assert else import pass
break except in raise
