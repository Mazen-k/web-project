Lists

Motivation

Given a sequence of numbers entered by user,how to:

  • Print them in reverse order?
  • Check if they are distinct?
  • Sort them?

list data structure

  • Use brackets [] to inistalize list in Python
  • Elements are sepetrated by commas
    • L1 = [1, 2, 3, 4, 5, 6]
    • L2 = ["Hello World!", 12, 3.14, False]
  • list is either
    • Homogenous:same data-type (like L1)
    • Heterogenous:mixed data-type (like L2)
  • The built-in len() gives the length of the list

list indexing

  • list elements can be accesed by specifying the index
    • Non-negative indices: starting from 0 and end at length -1
    • Negative indies: from right to left (reverse order)
  • Lists are mutable data structures meaning they can be modifed

More list functionalities

            
Lists have alot of built in functions some of which are:

            

                
                    

                        Function
                        Description
                    

                
                
                    

                        .append(x)
                        Addes x to the end of the list
                    

                    

                        .remove(x)
                        Removes first occurence of x
                    

                    

                        .count(x)
                        Returns the number of times x appears in the list
                    

                    

                        .index(x)
                        Returns the index of the first occurence of x
                    

                    

                        .pop(i)
                        Removes and returns the elemnt at index i
                    

                    

                        .insert(i , x)
                        Insterts item x at index i
                    

                    

                        .copy()
                        Returns a shallow copy of the list
                    

                    

                        .sort()
                        Sorts the list in ascending order
                    

                    

                        .reverse()
                        Reverses the order of the elements in the list
