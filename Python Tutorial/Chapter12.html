Tuples

tuple data type

  • Use parentheses() to initalize a tuple in Python
  • Elements are seperated by commas
    • T1 = (1, 2, 3, 4, 5, 6)
    • T2 = ("Hello World!", 12, 3.14, False)
    /ul>
  • tuple is either
    • Homogenous:same data-type (like T1)
    • Heterogenous:mixed data-type (like T2)
  • The built-in len() gives the length of the tuple
  • Tuples are immutable meaning that once intialized cannot be changed, i.e., read-only
