Functions

What is a function

  • A set of statements grouped together that perform a specific task
  • A function is characterized by its
    • name (usually describes the task being performed)
    • input parameters
    • return value
  • Function can return:
    • A value of any type(int,float,bool...)
    • Nothing, thus return value is NoneType

Built in functions

  • We have seen many functions so far
    • Input/Output: input(), print()
    • Type-casting: int(), bool()
    • Others: len(), type()
  • Other useful functions can be accsed from modules
    • Example: math module: ceil(), sqrt(), cos(), sin()
