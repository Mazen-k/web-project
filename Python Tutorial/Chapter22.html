Sets

Python set

  • A Python set is an object that stores a collection of data in the same way as a mathematical set
  • All items must be unique(no duplicates)
  • Set is unordered (insertion order not maintained)
  • Elements in a set can be of different data types
  • A set can be modified, but the elements of a set are immutable (since there is no indexing in set)

Creating a set

  • Method 2: a set can be defined with curly braces({}):
    • For example, x = {'x',q','u'}
    • if we call print x, we get:{'x,'q','u'}(notice how duplicates are removed)
    • However, x ={} creates an empty dictionary. Hence the only way to create an empty set is to call set ()

Elements in a set

Function Description
len(set) Returns the number of elements in the set.
add(element) Adds an element to the set
update(iterable) Adds multiple elements from an iterable to the set
pop() Removes and return a random element
clear() Clears all elements in the set
remove(element) Removes the specified element(Raises a KeyError exception)
discard() Removes the specified element(Does not raise an exception)

for loop, in, and not in operators

  • A for loop can be used to iterate over elements in a set
  • General format: for item in set
  • The loop iterates once for each element in the set
  • The in operator tests whether a value exists in a set
  • The not in operator tests whether a value does not exist in a set

Title

Concept Description
Finding the union of sets
  • Union of two sets: A set that contains all elements of both sets
  • Use the union method: set1.union(set2)
  • Or use the | operator: set1 | set2
  • Both techniques return a new set with the union of both sets
Finding the intersection of sets
  • Intersection of two sets: A set containing elements that exist in both sets
  • Use the intersection method: set1.intersection(set2)
  • Or use the & operator: set1 & set2
  • Both techniques return a new set with the intersection of the sets
Finding the difference of sets
  • Difference of two sets: A set containing elements in the first set but not the second
  • Use the difference method: set1.difference(set2)
  • Or use the - operator: set1 - set2
  • Both techniques return a new set with the difference of the sets
Finding the symmetric difference of sets
  • Symmetric Difference: A set with elements that are not shared by both sets
  • Use the symmetric_difference method: set1.symmetric_difference(set2)
  • Or use the ^ operator: set1 ^ set2
  • Both techniques return a new set with the symmetric difference of both sets
Finding subsets and supersets
  • Subset: Set A is a subset of Set B if all elements of A exist in B
  • Use the issubset method: setA.issubset(setB)
  • Or use the <= operator: setA <= setB
  • Returns True if A is a subset of B
  • Superset: Set A is a superset of Set B if A contains all elements of B
  • Use the issuperset method: setA.issuperset(setB)
  • Or use the >= operator: setA >= setB
  • Returns True if A is a superset of B
